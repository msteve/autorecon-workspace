# AutoRecon V2 - Enterprise Reconciliation System

Full-stack enterprise reconciliation platform with React frontend and Django backend.

## Repository Structure

```
autorecon-workspace/
├── frontend/           # React TypeScript frontend
├── backend/            # Django REST API (to be generated)
└── README.md
```

## Quick Start

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Access at: http://localhost:3000

### Backend (Coming Soon)

Django REST API for reconciliation operations.

## Features

- ✅ Data ingestion & file upload
- ✅ Rule-based reconciliation engine
- ✅ Exception management
- ✅ Maker-checker approvals
- ✅ Settlement processing
- ✅ GL posting & journal entries
- ✅ Reporting & analytics
- ✅ Audit trail & compliance
- ✅ Multi-user & role-based access

## Technology Stack

**Frontend:**
- React 18 + TypeScript
- Vite, TailwindCSS, Shadcn UI
- React Query, Zustand, React Router

**Backend (Planned):**
- Django + Django REST Framework
- PostgreSQL
- Celery + Redis
- Docker
